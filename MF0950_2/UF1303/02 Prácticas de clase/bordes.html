Bordes CSS

Estilo de borde

La propiedad border-style especifica qué tipo de borde mostrar.

Se permiten los siguientes valores:

  • dotted - Define un borde punteado.
  • dashed - Define un borde discontinuo.
  • solid - Define un borde sólido.
  • double - Define un doble borde.
  • groove - Define un borde ranurado 3D. El efecto depende del valor del color del borde.
  • ridge - Define un borde acanalado en 3D. El efecto depende del valor del color del borde.
  • inset - Define un borde de inserción 3D. El efecto depende del valor del color del borde.
  • outset - Define un borde de inicio 3D. El efecto depende del valor del color del borde.
  • none - No define bordes
  • hidden - Define un borde oculto.

La propiedad border-style propiedad puede tener de uno a cuatro valores (para el borde superior, el borde derecho, el borde inferior y el borde izquierdo).

Ancho del borde

La propiedad border-width propiedad especifica el ancho de los cuatro bordes.

El ancho se puede establecer como un tamaño específico (en px, pt, cm, em, etc.) o utilizando uno de los tres valores predefinidos: delgado, medio o grueso.

La propiedad border-width propiedad puede tener de uno a cuatro valores (para el borde superior, el borde derecho, el borde inferior y el borde izquierdo).

Color del borde

La propiedad border-color propiedad se utiliza para establecer el color de los cuatro bordes.

El color puede ser establecido por:

  • nombre: especifique un nombre de color, como "red"
  • Hex: especifique un valor hexadecimal, como "# ff0000"
  • RGB: especifique un valor RGB, como "rgb (255,0,0)"
  • transparente

La propiedad border-color puede tener de uno a cuatro valores (para el borde superior, el borde derecho, el borde inferior y el borde izquierdo). 

Si border-color no está configurado, hereda el color del elemento.

Bordes individuales

En CSS, también hay propiedades para especificar cada uno de los bordes (arriba, derecha, abajo e izquierda):

Si la propiedad border-style tiene cuatro valores:

  • estilo de borde: punteado sólido doble discontinuo;
    • el borde superior está punteado
    • el borde derecho es sólido
    • el borde inferior es doble
    • el borde izquierdo está punteado

Si la propiedad border-style tiene tres valores:

  • estilo de borde: punteado sólido doble;
    • el borde superior está punteado
    • Los bordes derecho e izquierdo son sólidos
    • el borde inferior es doble

Si la propiedad border-style tiene dos valores:

  • estilo de borde: punteado sólido;
    • Los bordes superior e inferior están punteados
    • Los bordes derecho e izquierdo son sólidos

Si la propiedad border-style tiene un valor:

  • estilo de borde: punteado;
    • los cuatro bordes están punteados

La propiedad border-style se utiliza en el ejemplo anterior. Sin embargo, también funciona con border-width y border-color.
