Etiquetas que definen una tabla

Una tabla HTML se define con la <table> etiqueta.

Cada fila de la tabla se define con la <tr> etiqueta. Un encabezado de tabla se define con la <th> etiqueta. Por defecto, los encabezados de las tablas están en negrita y centrados. Una tabla de datos / celda se define con la <td> etiqueta.


Etiqueta Descripción
<table> Define la estructura de una tabla
<tr> Define una fila en una tabla
<th> Define una celda de cabecera en una tabla
<td> Define una celda dentro de una fila en una tabla
<caption> Define el títula o leyenda de una tabla
<colgroup> Especifica en una tabla un grupo de una o más columnas para darle formato
<col> Especifica propiedades de columna para cada columna dentro de un elemento <colgroup>
<thead> Agrupa el contenido del encabezado en una tabla
<tbody> Agrupa el contenido del cuerpo en una tabla
<tfoot> Agrupa el contenido del pie de página en una tabla

La etiqueta <caption> define un título de tabla y debe insertarse inmediatamente después de la etiqueta <table>.

El atributo Scope

El atributo scope provee un mecanismo para indicar explicitamente a qué celdas afecta una celda de encabezado.

El atributo de ámbito especifica si una celda de encabezado es un encabezado para una columna, fila o grupo de columnas o filas. el atributo de alcance no tiene ningun efecto visual en los navegadores web normales, pero puede ser utilizado por los lectores de pantalla

El atributo colspan y rowspan

El atributo colspan define el número de columnas que debe abarcar una celda de encabezado. El atributo rowpan define el número de filas que debe abarcar una celda de encabezado.

El atributo colgroup y col

La etiqueta <colgroup> especifica un grupo de una o más columnas en una tabla para formatear. La etiqueta <colgroup> es útil para aplicar estilos a columnas completas, en lugar de repetir los estilos para cada celda, para cada fila.

La etiqueta <col> especifica las propiedades de columna para cada columna dentro de un elemento <colgroup>. La etiqueta <col> es útil para aplicar estilos a columnas enteras, en lugar de repetir los estilos para cada celda, para cada fila.

Agrupación por filas

HTML provee tres elementos que pueden ayudar a los autores a organizar la informción presentada en una tabla. Éstos son el <thead>, para agrupar las filas que presentan la información de encabezado, <tfoot> para contener a las filas que representan un pie o un resumen, y <tbody> para representar un bloque de filas que consiste en un cuerpo de datos.

Las etiquetas <thead>, <tbody> y <tfoot> se utilizan en conjunción con los tbody y tfoot elementos para especificar cada parte de una tabla (cabecera, cuerpo, pie de página).

La etiqueta <thead> debe usarse en el siguiente contexto: como elemento secundario de un elemento <table>, después de cualquier elemento <caption> y <colgroup>, y antes de cualquier <tbody>, <tfoot> y <tr> elementos.

Estas etiquetas permiten agrupar filas para darle estilos CSS.


<caption>Leyenda de tabla</caption>
<thead> <tr rowspan="2">
