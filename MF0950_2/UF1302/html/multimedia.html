HTML Multimedia

¿Qué es multimedia?

La multimedia viene en muchos formatos diferentes. Puede ser casi cualquier cosa que puedas oír o ver.
Ejemplos: imágenes, música, sonido, videos, discos, películas, animaciones y más.
Las páginas web a menudo contienen elementos multimedia de diferentes tipos y formatos.
Los elementos multimedia (como audio o video) se almacenan en archivos multimedia.
La forma más común de descubrir el tipo de un archivo, es mirar la extensión del archivo.
Los archivos multimedia tienen formatos y diferentes extensiones como: .swf, .wav, .mp3, .mp4, .mpg, .wmv y .avi.

Etiquetas de video

Antes de HTML5, un video solo se podía reproducir en un navegador con un complemento (como flash). La etiqueta <video> HTML5 especifica una forma estándar de incrustar un video en una página web.

Etiqueta Descripción
<video> Define un video o película.
<source> Define múltiples recursos de medios para elementos de medios, como <video> y <audio>.
<track> Define pistas de texto en reproductores multimedia.

Ejemplo de video

CÓDIGO HTML

<video width="320" height="248" controls preload="auto" loop muted poster="img/paisaje.png">
<source src="video/mov_bbb.mp4" type="video/mp4">
<source src="video/mov_bbb.ogg" type="video/ogg">
<source src="video/mov_bbb.webm" type="video/webm">
<track src="subtitulos_en.vtt" kind="subtitulos" srclang="en" label="English">
</video>


Etiquetas de audio

El atributo controls agrega controles de audio, como reproducir, pausar y volumen.

La etiqueta <source> le permite especificar archivos de audio alternativos entre los que el navegador puede elegir. El navegador utilizará el primer formato reconocido.

El texto entre las etiquetas <audio> y </audio> sólo se mostrará en los navegadores que no soportan el elemento <audio>.

Ejemplo de audio

CÓDIGO HTML

<audio controls>
<source src="audio/intro.ogg" type="audio/ogg">
<source src="audio/intro.mp3" type="audio/mpeg">
<source src="audio/intro.mp3" type="audio/wav">
Tu navegor no soporta el elemento audio.
</audio>

HTML Object

El <object>elemento es soportado por todos los navegadores.

El <object>elemento define un objeto incrustado dentro de un documento HTML.

Ejemplo de uso Object

Código HTML

<object width="400" height="50" data="swf/bookmark.swf"></object>



Nota: Se puede utilizar el estilos CSS "list-style-type" para especificar el tipo de viñeta. El atributo type en la etiqueta <ul> no se admite en HTML5. Los tipos de viñeta admintidas en CSS son: circle: círculo negro y relleno, disc: línea círcular, square: cuadro relleno, none: Este valor elimina la viñeta.

El <object>elemento define un objeto incrustado dentro de un documento HTML.

Ejemplo de uso Object

Código HTML

<iframe width="560" height="315" src="https://www.youtube.com/embed/MEVEUTnFzu8" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
