Etiquetas que citan el texto

Las abreviaturas de marcado <abbr> pueden proporcionar información útil a los navegadores, sistemas de traducción y motores de búsqueda. Utiliza el atributo title para definir el significado de la abreviación

La etiqueta de marcado <cite> hace referencia a un trabajp creativo como un libro, una película, un programa de televisión o una canción, no el nombre de una persona.

La etiqueta de marcado <blockquote> define una cita larga que se representa como un elemento de bloque. No genera automaticamente comillas arededor del contenido.

La etiqueta de marcado <q> define una cita corta en línea. Marca la cita a otros autores o documentos. Genera automaticamente comillas arededor del contenido.

Las estiquetas de marcadop <dfn> define un término y debe de contener el atributo title para la definición/explicación del termino dentro.

Etiquetas de código de computadora

  • La etiqueta <code> define un fragmento de código de ordenador.
  • La etiqueta <kbd> representa comandos de entrada del usuario.
  • La etiqueta <samp> representa la salida de un programa o sistema informático.
  • La etiqueta <var> se utiliza como expresión matemática o variable de la programación.

Etiquetas que separan el texto

La etiqueta <p> separa el texto, añadiendo automáticamente un espacio (margen) antes y después de cada elemento <p>.El navegador eliminará todos los espacios en blanco y las líneas adicionales cuando se visualiza la etiqueta <p>.

La etiqueta <br> es una etiqueta vacía, lo que significa que no tiene ninguna etiqueta final. Inserta un salto de línea (una nueva línea) sin comenzar un nuevo párrafo.

Un texto dentro de una etiqueta <pre> se muestra en una fuente de ancho fijo (generalmente Courier), y conserva ambos espacios y saltos de línea.
