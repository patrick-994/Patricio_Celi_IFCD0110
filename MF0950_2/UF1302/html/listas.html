Etiquetas de lista


Etiqueta Descripción
<ul> Define una lista desordenada.
<ol> Define una lista ordenada
<li> Define un elemento de lista
<dl> Define una lista de descripciones.
<dt> Define un término en una lista de descripción
<dd> Describe el término en una lista de descripción

Listas desordenadas

Una lista desordenada comienza con la etiqueta <ul>. Cada elemento de la lista comienza con la etiqueta <li>. La list-style-type propiedad CSS se utiliza para definir el estilo del marcador de elemento de lista.

Listas ordenadas

Una lista ordenada comienza con la etiqueta <ol>. Cada elemento de la lista comienza con la etiqueta <li>. El atributo type de la etiqueta <ol>, define el tipo de marcador de elemento de lista.

  1. type="1"
  2. type="A"
  3. type="a"
  4. type="I"
  5. type="i"

Listas de descripción

Una lista de descripción es una lista de términos, con una descripción de cada término. La etiqueta <dl> define la lista de descripción, la etiqueta <dt> define el término (nombre) y la etiqueta <dd> describe cada término.

<ul>
- Etiqueta HTML para definir una lista desordenada.
list-style-type
- Propiedad CSS para definir el marcador de elemento de lista.
<ol>
- Etiqueta HTML para definir una lista ordenada.
type
- Atributo HTML para definir el tipo de numeración.
<li>
- Etiqueta HTML para definir un elemento de lista.
<dl>
- Etiqueta HTML para definir una lista de descripción.
<dt>
- Etiqueta HTML para definir el término de descripción.
<dd>
- Etiqueta HTML para describir el término en una lista de descripción.
Las listas
- se pueden anidar dentro de otras listas.
Los elementos
- de la lista pueden contener otros elementos HTML.
float:left o display:inline
- Propiedad CSS para mostrar una lista horizontalmente
