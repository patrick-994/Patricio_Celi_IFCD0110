Formato HTML

El formato de texto, es el marcado que se aplica a un texto para añadirle datos de forma.

Al texto resultante se le conoce como texto formateado, textos con estilos o texto enriquecido

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios hojean sus páginas por sus encabezados. Es importante usar encabezados para mostrar la estructura del documento.

Los encabezados <h1> se deben usar para los encabezados principales, seguidos de los encabezados<h2>, luego los menos importantes <h3>, y así sucesivamente.

Nota: Use encabezados HTML solo para encabezados. No uses encabezados para hacer el texto GRANDE o en negrita.

<h1> Describe un encabezado de tipo 1

Encabezado tipo 1

 <h1>Esto es un encabezado de tipo 1</h1>
<h2> Describe un encabezado de tipo 2

 <h1>Esto es un encabezado de tipo 2</h1>

 <h3>Esto es un encabezado de tipo 1</h3>

Etiquetas que dan formato al texto

HTML también define etiquetas para definir texto con un significado especial.

HTML utiliza etiquetas para dar formato de salida, como el texto en negrita o cursiva.

Etiqueta Descripción Ejemplo Utilización
<b> Define un texto en negrita Este texto está en negrita <b>Este texto está en negrita</b>
<i> Define un texto en cursiva Este texto está en cursiva <i>Este texto está en cursiva</i>
<strong> Define un texto importante/resaltado Este texto es importante <strong>Este texto es importante</strong>
<em> Define un texto enfatizado Este texto está enfatizado <em>Este texto está enfatizado</em>
<small> Define un texto más pequeño Este texto se muestra más pequeño Este texto se muestra <small>más pequeño</small>
<sub> Define un texto en formato subíndice Este texto es subíndice de texto Este texto <sub>es subíndice</sub> de texto
<sup> Define un texto en formato superíndice Este texto es superíndice de texto Este texto <sup>es superíndice</sup> de texto
<ins> Define un texto insertado/agregado Este texto está agregado Este texto está <ins>agregado<ins>
<s> Define un texto tachado Este texto es un texto tachado Este texto es un texto <s>tachado<s>
<u> Define un texto subrayado Este texto está subrayado Este texto es un texto <u>subrayado<u>
<del> Define un texto borrado/eliminado Este texto está tachado eliminado Este texto está <del>tachado</del> eliminado
<mark> Define un texto marcado/subrayado Este texto está marcado Este texto está <mark>marcado</mark>

Nota: los navegadores se muestran <strong> como <b> y <em> como <i>. Sin embargo, hay una diferencia en el significado de las siguientes etiquetas: <b> y <i>define el texto en negrita y cursiva, pero <strong> y <em> significa que el texto es "importante".
